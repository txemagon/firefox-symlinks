Hur du hittar ämnet du söker

För att visa hjälptexten i det här fönstret, klicka på ämnena i sidofältet Innehåll. Klicka på ett ämnes plusikon triangel för att se underavdelningar.

För att söka efter att hjälpavsnitt, skriv in ett ord eller en fras i Sök-fältet för att se en lista på relevanta ämnen. Klicka sedan på det ämne du vill läsa om. Om sökningen inte ger några resultat, försök då att skriva in färre ord eller en annan kombination av ord. Sökresultaten visas automatiskt i sidofältet när du skriver. För att återigen visa innehållet i sidofältet, rensa Sök-sidofältet, eller tryck Esc.

För att se ytterligare information om det ämne du läser om, klicka på länkarna i Hjälp-fönstret på samma sätt som du gör med länkar på vanliga webbsidor. Du kan även gå bakåt och framåt bland de sidor du redan visat genom att använda Bakåt och Framåt-knapparna

Söktips

Hittar du inte det du söker? Här är några tips som kan hjälpa:

  • Var noga med att du söker efter sådant som rör &brandShortName;; det här är ingen sökning på webben.
  • Försök att bredda din sökning - var inte för specifik; termerna kan vara uttryckta på annat sätt än med de ord du söker på.
  • Undvik för breda sökningar. Ett vanligt ord som ”sida” kan t.ex. ge fler sökresultat än vad som är önskvärt.
